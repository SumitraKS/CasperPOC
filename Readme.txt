Details of the files are as follows.

Index.jsp
This is the home page. It has a username, password, login button, and links to w3schools and casperjs.org.

Tests - Folder
This foder contains the CasperJS test scripts to test the home page and the links.

01Home.js - tests the index.jsp
02Link1.js - tests the w3schools page
03Link2.js - tests the casperjs.org

The scripts are only a POC to see how the unit testing and integration testing can be done with CasperJS. 
As part of the unit test the components in the individual pages are tested.
As part of the integration test the existance and the landing pages of the links are tested
